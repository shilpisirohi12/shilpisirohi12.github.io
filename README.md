# Resume Creation Using Claude Code

Created static resume website using Claude Code which can be deployed as github pages.

## CLAUDE.md

This file is the mind which is used by claude to create the application. This is created later once the application is created by running /init command in claude code. This is used to provide context to claude to create the application.

## Benefit of creating CLAUDE.md using /init

These is the best way to create a CLAUDE.md file. It will create a CLAUDE.md file which is specific to the application and will provide the best context to claude to create the application. This will also optimise the token usage.

## How to create CLAUDE.md using /init

1. Run /init command in claude code.
2. Follow the instructions to create the CLAUDE.md file.
3. The CLAUDE.md file will be created in the root directory of the application.

## How to use CLAUDE.md

1. Open CLAUDE.md file in claude code.
2. Read the file and understand the application.
3. Follow the instructions to create the application.
4. The CLAUDE.md file will be updated as the application is created.

## How to update CLAUDE.md

1. Open CLAUDE.md file in claude code.
2. Read the file and understand the application.
3. Follow the instructions to update the application.
4. The CLAUDE.md file will be updated as the application is updated.

## NOTES

CLAUDE-manual.md - This file contains the manual thought process to create the application. This is used to document the thought process and can be used to recreate the application in future.
